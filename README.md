# Calculator
Simple command-line calculator in C.
Supports addition, subtraction, multiplication, and division.

# Calculator_1
Simple operation mode
Users input 2 seperate numbers and does a simple operation of +,-,* or /.

# Calculator_2
Now able to accept a single line expression
Operator need to be with space for the code to work well

# Calculator_3
Can now solve any expression format
Space not necessary for code to work well
Operators order of priority is implemented (BODMAS)

# Calculator_4
Handling of Unary minus / negative numbers
